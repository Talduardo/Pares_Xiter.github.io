# 🌐 Landing Page – Pares_Xiter

Este repositório contém o **front-end de uma landing page** desenvolvida para atender a uma **demanda real de uma loja**. O projeto foi criado com foco em apresentar produtos e serviços de forma clara, moderna e responsiva, funcionando como a porta de entrada digital da marca.

---

## 📂 Estrutura do Projeto:

- **index.html** → Estrutura principal da landing page.  
- **style.css** → Estilos customizados para layout, cores e responsividade.  
- **photos/** → Imagens utilizadas na página.  
- **README.md** → Documentação do projeto.  

---

## 🎯 Objetivos:

- Criar uma landing page **profissional e responsiva**.  
- Garantir **boa experiência do usuário** em diferentes dispositivos.  
- Atender a uma **demanda real de negócio**, funcionando como vitrine digital da loja.  

---

## 🚀 Funcionalidades:

- Estrutura simples e objetiva para destacar produtos/serviços.  
- Layout responsivo para desktop, tablet e mobile.  
- Uso de imagens otimizadas para melhor performance.  
- Design moderno com paleta de cores personalizada.  

---

## 🛠️ Tecnologias Utilizadas:

- **HTML5** → Estrutura semântica e acessível.  
- **CSS3** → Estilização responsiva e moderna.  
- **Imagens otimizadas** → Recursos visuais para destacar a marca.
